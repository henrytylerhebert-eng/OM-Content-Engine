# Enrichment Layer

The enrichment layer adds derived fields that help with storytelling, segmentation, and matching.

Examples:

- audience tags
- proof tags
- content readiness
- mentor matching helpers
- rule-based ecosystem segments
- content intelligence records for storytelling and promotion

These helpers should add context, not change the operational truth coming from the source.

Current first-pass modules include:

- `audience_segments.py` for narrow expertise and mentor filters
- `ecosystem_segments.py` for reporting and outreach segments
- `content_intelligence.py` for founder, mentor, partner, and organization content packaging signals
